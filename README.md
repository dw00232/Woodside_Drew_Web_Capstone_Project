# Web-Development-Project
GAME REVIEW HUB

This is my Web Development Capstone project. I built this site to provide honest, community-driven reviews and news for indie games. My goal is to help smaller developers get noticed by casual gamers.

Installation:

No installation is required. You can view the project code here on GitHub or visit the live website using my deployment link.

Usage:

1. Homepage: View our mission statement and site goals.

2. Game Reviews: Look at scores and click screenshots to see different views.

3. News: See the release calendar and use the slide or highlight features.

4. Community: Use the form for indie developer game submissions.
